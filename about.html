About Link Dock

Link Dock

Version 1.0.0

Zero-Backend State

Link Dock completely bypasses the need for a database. Your links, name, and profile configurations are fully compiled into a base64 encoded parameter stored securely inside the URL string itself.

Animated UI Core

Engineered with responsive cubic-bezier transitions, featuring smooth bottom-sheet action menus for mobile, drag-and-drop sorting for desktop, and instantaneous Dark/Light theme switching.

Dynamic Auto-Favicons

The Link Dock renderer securely pulls high-resolution favicons directly from the destination URL domains to automatically match the aesthetic of your external links.
